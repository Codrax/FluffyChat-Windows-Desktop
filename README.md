# FluffyChat-Windows-Desktop
A desktop verison of the FluffyChat client for Windows with a Microsoft Edge2 WebView, with support for Windows Action Center notifications. 

### Features
- The entire fluffy chat client
- Access your chats at anytime, even without an internet connection
- Start FluffyChat with your computer automatically
- Minimize the app into the system tray when closed (also when started with your computer)
- Notifications for new messages
- Notification for recieving a call
- Automatic updates


To access most settings and UIs specific to the app, right-click on the application's titlebar and press "Settings", or access the settings pane by right-clicking the system tray icon.
